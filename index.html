Search for Ada/SPARK on the Web

Search for a phrase in Google:

Search for the Ada language —or its subset SPARK— related things on the Web. This search engine is powered by Google Programmable Search Engine. After showing results of your query, you can sort them by relevance or date. The search engine is customized to find only Ada/SPARK related things, thus queries like "aws" return only "Ada Web Server" related pages.

Google Fu moves:
  • Exact phrase: "use quotes around"
  • All of the words: just plain words
  • One of the words (separate by OR): this OR that OR word
  • Without the words (start the word with the minus sign): -not -this -words
  • Numbers range: (the range with units) 10..20 kg (just the range) 2011..2014
  • Only the selected page: site:example.com
  • Older than the selected date: before:YYYY-MM-DD
  • Newer than the selected date: after:YYYY-MM-DD

Of course, you can merge any of these moves to create a combo :)

Search for Ada code in searchcode:

Search for a phrase in several source code forges, like Gitlab or Bitbucket. This search engine is powered by searchcode. The search is customized to look only for code written in Ada.

Search for Ada code in Alire crates:

Search for any source code pattern found in any of the Alire crates. This search engine is powered by search.ada-lang.io.

Search for Ada repositories in GitHub:

Search for any phrase found in GitHub.com Ada repositories. Keep "lang:ada" on the query to look only for code written in Ada.

Search for Ada projects in Sourceforge:

Search for any pattern found in the Sourceforge Ada directory.

Search for a phrase on ada-auth.org search engine:

Search for the Ada language —or its subset SPARK— related things on the Web. This search engine is powered by the Ada Conformity Assessment Authority.

Search for a phrase on the Ada Programming wikibook:

Search for the any phrase on the Ada Programming wikibook.

This site includes OpenSearch description files for all the search engines. That means that you can add them as alternative search plugins to your browser.
